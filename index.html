Команди Git

Base

  • git config --global user.name blabla - задати ім'я користувача
  • git config --global user.email blabal@bla - задати електронну адресу користувача
  • git init - створення git-репозиторія
  • git status - показує стан файлів
  • git add - додає вміст для нового коміту
  • git commit - робить коміт
  • git log - журнал комітів
  • git checkout - переключатися між гілками і перегляд даних з гілки
  • git branch - показує наявні гілки, дає можливість додавати нові, видаляти і змінювати назву
  • git push - з'єднання з віддаленим репозиторієм
  • git pull- комбінація команд git fetch і git merge, тобто спочатку збираються зміни з вказаного віддаленого репозиторію, а тоді зливає з даною гілкою
  • git merge - злиття гілок в одну
  • git clone - клонувати віддалений репозиторій

.gitignore - в цьому файлі записуємо все, що має ігнорувати git

Advanced

  • git revert - скасувати існуючий коміт
  • git reset - скасувати останній коміт
  • git rebase - об'єднати коміти з однієї гілки з іншою гілкою/li>
  • git cherry-pick - застосовує зміни, внесені деяким комітом і застосовує їх для повторно для гілки
  • git stash - дозволяє відкласти зміни
